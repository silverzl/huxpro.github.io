   S执行次数：(n+3)(n-2)/2
   
### 应用题：

1. 有如下递归函数，请分析其时间复杂度。
```
fact(int n)
{
  if（n<=1）  return 1;  (1)
  else  return (n*fact(n-1));  (2)
}
```
设fact(n)的运行时间函数是T(n),语句(1)的运行时间为O(1),语句(2)的运行时间为T(n-1)+O(1),所以

```
T(n)=   O(1)  n<=1
T(n-1)+O(1)  n>1

所以,T(n)= O(1)+T(n-1)  =2*O(1)+T(n-2)  =…
     =(n-1)*O(1)+T(1)  =n*O(1)  =O(n)
     
```
